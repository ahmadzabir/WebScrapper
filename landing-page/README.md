# Desktop Download Landing Page

This folder contains a static one-button download page for the Windows installer.

## Publish Options

- GitHub Pages (recommended): publish this folder as static site content.
- Netlify/Vercel static hosting: drag and drop the folder.

## Update Download Link

The main button currently targets:

`https://github.com/ahmadzabir/WebScrapper/releases/latest/download/WebScrapperDesktop-Setup.exe`

Make sure each release includes an asset named exactly:

`WebScrapperDesktop-Setup.exe`

If your repo URL changes, update the link in `index.html`.
