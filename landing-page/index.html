🌐 Web Scraper

Upload a CSV of URLs and scrape clean text (and optional AI summaries). No installers—use in the browser or run locally with two commands.

View on GitHub How to run

Use online

Deploy to Streamlit Cloud (free), connect your GitHub repo, and use the app in the browser. See the repo README for deploy steps.

Run locally

You need Python 3.10+. In the project folder run:

pip install -r requirements.txt
streamlit run python-scraper.py

Then open the URL shown (e.g. http://localhost:8501) and upload your CSV.
